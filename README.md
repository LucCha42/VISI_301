# Goodoo

Started as a second year video game project, Goodoo is now a WIP side project.

## Requirement

Python 3.9 or above

## Dependency

- [Pygame](https://pypi.org/project/pygame/) ```pip install pygame```

## Execution example

```python goodoo.py resources/maps/map1.jpg```

## Key Bindings

left  ```left``` / ```q```<br/>
right ```right``` / ```d```<br/>
jump ```up``` / ```z``` / ```space```<br/>
pick ```shift``` / ```r```<br/>
action ```left click```<br/>
